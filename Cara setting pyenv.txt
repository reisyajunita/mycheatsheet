Cara merubah pyenv python version
1. Python Global
- buka cmd atau powershell (biasanya cmd)
- install versi python yang dibutuhin (kl udah ada gausah langsung ke next step), ketik:
	`pyenv install 3.10.11`
- ketik `pyenv global 3.10.11`
- ketik `python --version` (ngecek udah berubah apa blm)

	`pyenv install 3.12.7`
- ketik `pyenv local 3.12.7`
- ketik `python --version`
